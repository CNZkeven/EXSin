我大抵是太过于**轻信**了。

那猫初来的时候，**毛色确是极好的**，**油光水滑**，在暗夜里也似能泛出微光来。我原以为，在这人世的荒凉里，我与它总算得上是**志趣相投**，能在这逼仄的小屋里，彼此做个**依偎**，平平安安地度过些流年。

然而，世间的事，大抵是不遂人愿的。我须得整日埋头于那些枯燥的营生里，好挣得几口粗食。猫呢，终究是生着四条腿的，向往着墙头上的月光和外面的春风。它总是要出去巡游的，时常是不见踪影。久而久之，我在这头，它在那头，竟成了所谓“**异地**”的局面。

终于有一日，它回到我面前，喵喵地叫着，做出十分**凄楚**的模样，大意是说：这异地的苦楚，它是再也熬不住了，凄清太甚，不如**就此别过**。我看着它那似乎有些委屈的胡须，心里自然是有些**隐痛**的。但勉强留住的，终究是不中用的，我便也只叹了一口气，伤心地由它去了。我以为，这不过是世间无数无常中的一个，是无可奈何的境遇造就的悲哀罢了。

直到过了些时日，周围的闲人们——他们向来是目光如炬的——特意跑来告诉我，说早在我与那猫“异地”之时，它便已在别家的灶台下**偷了腥**，且与另一只野物**耳鬓厮磨**得十分熟稔了。他们说得绘声绘色，仿佛那偷来的鱼腥味，正挂在他们自己的嘴边。

听到这话的时候，我的头脑里恍惚了一下，像是在极遥远的天际，沉闷地滚过了一阵雷。
然而，我摸了摸自己的心口，竟**没有一丝一毫的愤怒**。

这原是我早就预料到的，或者说，是我潜意识里早有觉察，只是未曾忍心去揭穿的把戏。我并不因为它急匆匆地、无缝衔接似的寻了下一个宿主而感到悲哀。这世道本就寒冷，禽兽**趋利避害**，去寻一个更暖和的窝，更丰足的鱼骨头，本是它的**本能**。我向来是不屑于用人类的所谓“**忠诚**”去苛责一只猫的。

我所感到非常遗憾，非常失望的，
是那一番**谎言**。

它分明是早已做了决断，却偏要借着“受不了异地”的名目，装出一副**楚楚可怜**的牺牲者模样来。它本可以坦诚地说一句“我有了更好的去处”，却非要用一块**虚伪的遮羞布**，掩盖它内心的**背叛与算计**。

这种**不坦诚**，这种精心粉饰的谎言，
才是真正令人发冷的东西。

我看着窗外灰蒙蒙的天，心里只觉得滑稽。我以为我们是**志同道合**的旅伴，哪知道从一开始，我们的皮毛下就藏着**截然不同的骨血**。**价值观的参差**，原来从它踏进门槛的那一刻起，就已经注定了今日的局。

远处的雷声终究只是一阵闷响，并没有真的劈下来，因为实在不值当。那只毛色不错的猫，终于在我记忆里**褪了色**，只留下一地斑驳的**虚伪**。**罢了**，随它去吧。
